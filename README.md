# Dailymotion-Client

Welcome to the Dailymotion-Client wiki!

* upload videos from local H.D
* remote upload by direct url
* view videos in (144-240-380-480-720)
* generate direct video links
* share videos
* edit video settings
* add video tags
* delete video
