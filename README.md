# Dailymotion-Client

Welcome to the Dailymotion-Client wiki!

* upload local videos
* upload remote videos
* whatch videos in (144-240-380-480-720)
* generate direct mp4 video links [(Video)](https://www.dailymotion.com/video/x7bdcfq)
* share videos
* edit video settings
* add video tags
* delete video
* Send public video from dailymotion account to supported cloud services with ability to specify transferred video resolution
